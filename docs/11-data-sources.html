Chapter 1 Data sources

The aim of this chapter is to present an overview of teh different type of data sources that can be use

1.1 Minimum requirement

In any case the source should provide a list of texts described by four compulsory fields

  • id : identification of the text (num)
  • source : code of the source (char)
  • date : date of publication or reception of the text (char)
  • lang : language of the text (char)

Other fields can be added as option :

  • url : when the text is linked to a webpage (char)
  • title : in the case where the title is separated from the text (plain text)
